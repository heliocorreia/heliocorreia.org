Mesmo com o surgimento do mouse, as teclas de atalho continuam mais forte do que nunca. Um amigo costuma dizer: "Só quem salva é Jesus Cristo e o Ctrl+S</em>". É comum encontrar nos teclados atuais, uma tecla aceleradora conhecida por Windows Key. Sua função é agilizar as tarefas rotineiras, por exemplo:

  • WinKey: Ativa o menu Iniciar</li>
  • WinKey + E: Abre o Windows Explorer</li>
  • WinKey + M: Minimiza todas as janelas. Talvez esta seja a mais amada por todos os que costumam ver pornografia no ambiente de trabalho.</li>
    </ul>
    Dentre todas as disponíveis, a combinação WinKey + R (Executar Programa) abre um leque ainda maior de possibilidades, ao realizar operações que com o mouse levaria uma eternidade. O site FixMyXP reúne uma lista de 99 combinações, que é de grande valia para qualquer usuário MS Windows. Eis algumas delas:

    • calc: Abre a calculadora</li>
    • services.msc: Abre o painel dos serviços do sistema</li>
    • shutdown: Desliga o computador</li>
      </ul>
      Para utilizá-las é bem simples:

      1. WinKey + R. Irá abrir uma pequena janela, "Executar".</li>
      2. Digite: "winmine"</li>
        </ol>
        Pronto, este pode ser seu novo atalho sempre que for jogar o campo minado.
