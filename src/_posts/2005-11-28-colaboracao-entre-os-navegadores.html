Representantes dos navegadores Internet Explorer, Firefox, Opera, e Konqueror se reuniram para discutir formas mais eficazes para inibir os ataques aos usuários menos atentos. Dentre as soluções apontadas, algumas já estão em funcionamento, ou em estágio beta de desenvolvimento, como são os casos do Firefox e do Internet Explorer 7, respectivamente.</p>

Vejamos o que será bastante comum nas próximas versões de nossos navegadores preferidos:</p>

  • Janelas popups indicarão o endereço real da página exibida.</li>
  • Páginas seguras indicação um pequeno cadeado na barra de endereço.</li>
  • Barra de endereço colorida: Amarelo para páginas seguras, verde para as confiáveis e vermelho para as consideradas maliciosas.</li>
    </ul>

    As formas como cada navegador apresentará estas e outras informações podem se diferenciar um pouco, mas devem ser tão intuitivas como os tão conhecidos botões Avançar e Voltar.</p>

    O interessante desta virada de mesa é que o Internet Explorer, que antes utilizava a vantagem do monopólio para ditar os padrões, agora está interagino com a comunidade, buscando a padronização que os concorrentes tentam seguir a todo o custo.</p>

    Futuramente espera-se que a padronização seja compatível entre os principais navegadores. Ficará a cargo do usuário escolher aquele que é capaz de oferecer as melhores ferramentas para a necessidade de cada ambiente. Atualmente, para mim, apenas um, dentre os citados acima, é capaz de se adaptar a qualquer tipo de usuário.
    </p>
