Estou para conhecer um notebook, com softwares originais de fábrica, que não acompanhe vários e vários "programas amigáveis". Do que adianta ter uma super máquina, se todo este lixo ocupa boa parte do processamento e da memória? Navegando pela net, encontrei um aplicativo muito útil para usuários do Windows.

"nLite é uma ferramenta para remoção permanente de componentes do Windows, e configuração de instalações pré-configuradas. Após a remoção, existe uma opção para gerar imagens botáveis para gravação ou máquinas virtuais. Com nLite você poderá ter instalações do Windows não inclui, nem mesmo no cd, componentes indesejáveis."


O nLite está disponível para Windows 2000, XP x86/x64 e 2003 x86/x64 em qualquer idioma, assim como o Windows Vista. Para a instalação, será necessário que o .NET Framework 1.1 esteja instalado. Recomendo o download.
