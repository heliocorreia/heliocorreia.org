Utilizando Javascript, o Nicholas Roussos publicou uma dica simples, mas bastante útil: Alterar o tamanho da janela do navegador para simulando uma mudança de resolução.

  • 640x480
  • 800x600
  • 1024x768


O mesmo princípio pode ser aplicado para definir posicionamento:

  • Dock Left
  • Dock Right
  • Dock Top
  • Dock Bottom


Para salvar nos favoritos, basta apertar com o botão direito, e escolher adicionar. Para ficar organizado, sugiro por os itens dentro de uma pasta.
