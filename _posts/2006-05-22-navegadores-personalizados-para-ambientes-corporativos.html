Até pouco tempo atrás os provedores (UOL, AOL, Terra etc) forneciam um "kit de acesso", com os principais programas para acessar a Internet. Dentre eles, havia uma versão personalizada (leia-se prostituída) do Internet Explorer. Mais dia, menos dia, todos acabaram se tornando um componente em alguma obra de artesanato.

Mas vendo bem, estamos em outros tempos. Conforme a necessidade, instalamos plugins, ajustamos as preferências, organizamos favoritos. O navegador não é mais um simples programa para acessar sites, mas sim uma plataforma de aplicações web. Os profissionais de TI sentem na pele a dificuldade para manter os programas, plugins e configurações sempre atualizados.

Uma boa forma de agilizar este processo é modificar a instalação padrão dos navegadores. Atualmente estas configurações estão bem flexíveis. Veja como personalizar seu navegador favorito:

  • Availability of browser administration kits (.PPT)
  • Firefox 1.5 CCK (Client Customization Kit)
  • Internet Explorer Administration Kit (IEAK)
  • Opera Composer


Acho válido lembrar que estas soluções são voltadas para o ambiente Windows. No caso do Safari (Mac OS X) é possível utilizar os recursos do Apple Script para ajudar na configuração.
