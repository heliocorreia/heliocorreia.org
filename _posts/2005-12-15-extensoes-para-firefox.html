O site Life Hacker publicou um conjunto de extensões recentes:

  • Del.icio.us bookmarks
    Integração com o serviço de bookmark social, del.icio.us

  • FoXpose
    Exibe as Tabs abertas, em janelas lado a lado

  • Tab Preview
    Exibe uma miniatura, se o mouse estiver em cima da tab

  • Tab Sidebar
    Exibe miniaturas das páginas abertas, na barra lateral (sidebar)

  • Restart Firefox
    Reinicia o navegador com apenas um clique

  • Cacheout
    Veja a versão em cache de um site indispoível

  • Gmail File Space
    Acessa o Gmail como se fosse um cliente FTP, para armazenar arquivos

  • Xihna Here!
    Transforma as caixas de texto em um editor HTML WYSIWYG
