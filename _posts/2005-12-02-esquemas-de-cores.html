"Cor é um aspecto físico da natureza. A cor de um material é determinada pelos comprimentos de onda dos raios luminosos que as suas moléculas constituintes refletem. Um objeto terá determinada cor se não absorver justamente os raios correspondentes à freqüência daquela cor." (Cor, Wikipedia)

Cada cor possui a característica de transmitir uma mensagem diferente. Elas são capazes de influenciar o entendimento de como as coisas funcionam. O que é certo, o que é errado, o que é permitido, o que está disponível etc.

A fase de desenvolvimento de um esquema de cores, para uma ilustração ou interface de uma aplicação, é crítico. As corem devem estar em harmonia com cada elemento disponível. Isto não é uma tarefa nada fácil. Acredito que este tenha sido um dos fatores pelo qual sites catalogam o esquema de cores de outros sites. Também é possível criar o seu próprio esquema de cores.

Um fato que tenho percebido é a preocupação na criação de um esquema: criar uma forma acessível também aos deficientes visuais. O esquema ideal seria aquele universal, percebido por todos, mas algo quase impossível de se alcançar. Mas é nosso dever tentar reduzir a dificuldade na interpretação das cores.
